Working of Vigenère Cipher

Vigenère Cipher

Overview

The Vigenère Cipher is a polyalphabetic substitution cipher invented by Blaise de Vigenère in the 16th century. It encrypts alphabetic text using a keyword, which is repeated to match the length of the plaintext. Each letter of the plaintext is shifted based on the corresponding letter of the keyword, using the Vigenère square.

Vigenère Square

Encryption Formula

The encryption formula for the Vigenère Cipher is:

Ci = (Pi + Ki) mod 26

Where:

  • Ci is the i-th character of the ciphertext
  • Pi is the i-th character of the plaintext
  • Ki is the i-th character of the key

Decryption Formula

The decryption formula for the Vigenère Cipher is:

Pi = (Ci - Ki + 26) mod 26

Encryption Steps

  1. Choose a keyword and repeat it to match the length of the plaintext.
  2. For each letter in the plaintext, find the corresponding letter in the keyword.
  3. Use the Vigenère square to find the intersection of the plaintext letter's row and the keyword letter's column. This is the ciphertext letter.
  4. Repeat steps 2-3 for each letter in the plaintext.

Decryption Steps

  1. Repeat the keyword to match the length of the ciphertext.
  2. For each letter in the ciphertext, find the corresponding letter in the keyword.
  3. Locate the keyword letter's row in the Vigenère square and find the ciphertext letter in that row.
  4. The column heading of the ciphertext letter is the plaintext letter.
  5. Repeat steps 2-4 for each letter in the ciphertext.

Example

Let's encrypt the message "HELLO WORLD" using the keyword "SECRET".

Plaintext: H E L L O W O R L D
Keyword: S E C R E T S E C R
Ciphertext: Z I N E S P Q V N Q
