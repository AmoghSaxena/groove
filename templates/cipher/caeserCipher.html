Caesar Cipher

In cryptography, a Caesar cipher, also known as Caesar's cipher, the shift cipher, Caesar's code, or Caesar shift, is one of the simplest and most widely known encryption techniques. It is a type of substitution cipher in which each letter in the plaintext is replaced by a letter some fixed number of positions down the alphabet. For example, with a left shift of 3, D would be replaced by A, E would become B, and so on. The method is named after Julius Caesar, who used it in his private correspondence.


The encryption step performed by a Caesar cipher is often incorporated as part of more complex schemes, such as the Vigen√®re cipher, and still has modern application in the ROT13 system. As with all single-alphabet substitution ciphers, the Caesar cipher is easily broken and in modern practice offers essentially no communications security.

Example:
The transformation can be represented by aligning two alphabets; the cipher alphabet is the plain alphabet rotated left or right by some number of positions. For instance, here is a Caesar cipher using a left rotation of three places, equivalent to a right shift of 23 (the shift parameter is used as the key):

Plain: ABCDEFGHIJKLMNOPQRSTUVWXYZ
Cipher: XYZABCDEFGHIJKLMNOPQRSTUVW

When encrypting, a person looks up each letter of the message in the "plain" line and writes down the corresponding letter in the "cipher" line.

Plaintext: THE QUICK BROWN FOX JUMPS OVER THE LAZY DOG
Ciphertext: QEB NRFZH YOLTK CLU GRJMP LSBO QEB IXWV ALD

Complexity
Time: O(|n|)

Caesar Cipher

Overview

The Caesar Cipher, also known as the shift cipher, is one of the simplest and most widely known encryption techniques. It is a type of substitution cipher where each letter in the plaintext is replaced by a letter some fixed number of positions down the alphabet. For example, with a shift of 3, A would be replaced by D, B would become E, and so on.

History

The Caesar Cipher is named after Julius Caesar, who reportedly used it to communicate with his officials. It is a simple example of a substitution cipher and has been used historically for basic encryption.

Encryption Formula

The encryption can be represented using modular arithmetic by first transforming the letters into numbers, according to the scheme, A = 0, B = 1,..., Z = 25. The encryption of a letter x by a shift n can be described mathematically as:

En(x) = (x + n) mod 26

Decryption Formula

Decryption is performed similarly to encryption, except we subtract the shift value instead of adding it:

Dn(x) = (x - n) mod 26

Encryption Steps

  1. Convert each letter of the plaintext to its corresponding numeric value (A = 0, B = 1, etc.)
  2. Add the shift value to each numeric value
  3. Take the result modulo 26 (to wrap around the alphabet)
  4. Convert the numeric values back to letters

Decryption Steps

  1. Convert each letter of the ciphertext to its corresponding numeric value
  2. Subtract the shift value from each numeric value
  3. Take the result modulo 26 (to wrap around the alphabet)
  4. Convert the numeric values back to letters

Example

Let's consider the plaintext "HELLO" and a shift value of 3.

Encryption:

  • H (7) + 3 = 10 mod 26 = 10 -> K
  • E (4) + 3 = 7 mod 26 = 7 -> H
  • L (11) + 3 = 14 mod 26 = 14 -> O
  • L (11) + 3 = 14 mod 26 = 14 -> O
  • O (14) + 3 = 17 mod 26 = 17 -> R

The ciphertext is "KHOOR".

Decryption:

  • K (10) - 3 = 7 mod 26 = 7 -> H
  • H (7) - 3 = 4 mod 26 = 4 -> E
  • O (14) - 3 = 11 mod 26 = 11 -> L
  • O (14) - 3 = 11 mod 26 = 11 -> L
  • R (17) - 3 = 14 mod 26 = 14 -> O

The decrypted plaintext is "HELLO".
