Working of SHA256 Hash

SHA256 Hash

Overview

SHA-256 (Secure Hash Algorithm 256-bit) is a cryptographic hash function that produces a 256-bit (32-byte) hash value. It is a member of the SHA-2 family of hash functions and is widely used for data integrity and security purposes.

SHA-256 Formula

The SHA-256 hash function can be expressed as:

H(M) = SHA-256(M)

Where:

  • H(M) is the hash value of message M
  • SHA-256 is the SHA-256 hash function
  • M is the input message

SHA-256 Steps

  1. Padding: Pad the input message with a single 1 bit, followed by zeros until the length is 448 bits (56 bytes) short of a multiple of 512 bits (64 bytes). Append the original message length as a 64-bit big-endian integer.
  2. Parsing: Parse the padded message into 512-bit (64-byte) blocks.
  3. Initialize hash values: Initialize eight 32-bit hash values (a, b, c, d, e, f, g, h) with predefined constants.
  4. Process blocks: For each 512-bit block:
    1. Prepare the message schedule.
    2. Initialize working variables.
    3. Perform 64 rounds of compression.
    4. Update the hash values.
  5. Output: Concatenate the final hash values (a, b, c, d, e, f, g, h) to produce the 256-bit hash output.
